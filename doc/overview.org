* Overview

This repository is a personal [[https://guix.gnu.org/][GNU Guix]] configuration, structured as a
custom Guix channel.  It covers:

- A full =operating-system= declaration for VM-based testing.
- A =home-environment= for the desktop (Sway/Wayland stack, Emacs, etc.).
- Custom channel packages not yet in upstream Guix.
- Shared home service definitions reusable across machines.

** Design principles

1. *Single source of truth* — all shared definitions live in =systems/common.scm=.
   Entry points (=desktop.scm=, =vm.scm=) only import and use them.

2. *Reproducibility* — =channels.scm= pins upstream Guix to a specific commit.
   Rust packages vendor all crates via companion =-crates.scm= files so builds
   are fully offline-capable once the store is populated.

3. *Channel-as-code* — custom packages are first-class Guix channel packages
   (module name =(config packages ...)=), not one-off local expressions.

4. *Home environment first* — user-facing configuration (Sway, Waybar,
   Emacs, wallpaper, audio) lives in the =home-environment=, not in system
   services, following Guix's separation of system and user concerns.
