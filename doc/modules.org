* Modules

** =(systems common)= — =systems/common.scm=

The central shared module.  All definitions intended for reuse across
machines are defined and exported here.  Import it with:

#+BEGIN_SRC scheme
(use-modules (systems common))
#+END_SRC

*** Exports

| Symbol | Type | Description |
|--------|------|-------------|
| =%my-keyboard-layout= | =keyboard-layout= | =us,bg= with phonetic Bulgarian, =Win+Space= toggle |
| =%primary-user= | =user-account= | =ivand=, groups: =wheel audio video input netdev= |
| =%base-packages= | list | System-wide CLI packages appended to Guix's =%base-packages= |
| =%my-base-services= | list | =network-manager= + =wpa-supplicant= (real hardware only) |
| =%guix-service= | =service= | Guix daemon (=guix-service-type=, default config) |

*** =guix-home-service-for=

A convenience procedure that wraps =guix-home-service-type= for a single user.

#+BEGIN_SRC scheme
(guix-home-service-for "ivand" %vm-home-environment)
;; expands to:
;; (service guix-home-service-type
;;          (list (list "ivand" %vm-home-environment)))
#+END_SRC

Used in =systems/vm.scm= to wire the home environment into the OS at boot.

** =(systems desktop)= — =systems/desktop.scm=

Minimal entry point for =guix home reconfigure= on the real machine.
Contains only:

#+BEGIN_SRC scheme
(use-modules (systems common))
%desktop-home-environment
#+END_SRC

*Usage:*
#+BEGIN_SRC sh
guix home reconfigure -L . systems/desktop.scm
#+END_SRC

** =(systems vm)= — =systems/vm.scm=

Full =operating-system= for QEMU VM testing.  Inherits
=%desktop-home-environment= and adds a =home-bash-service-type= stanza to
auto-start Sway on tty1.  Wires the home environment into the OS at boot
via =guix-home-service-for=.

Key differences from a real machine:

- Uses =dhcpcd-service-type= instead of =network-manager-service-type=.
- Empty user password (PAM allows empty passwords).
- =%wheel NOPASSWD: ALL= in sudoers.
- Virtio kernel modules in =initrd-modules=.
- Sway auto-starts from =.bash_profile= instead of a display manager.

See =systems/vm.scm= for full inline usage instructions.
